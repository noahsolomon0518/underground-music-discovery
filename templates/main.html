Spotify Insight Tool

This app uses the spotify API related artist feature to help you find your next favorite artist.

Learn more below.

Related Artist Finder

Uses spotify's related artist feature to recursively find artist similar to inputted artist. Can view table of related artist with popularity values (0-100) and number of followers. Option to download as CSV file. Go here

Related Artist Playlist Generator

Creates a playlist based on the related artist tree of an inputted artist. Can condition on popularity and number of followers to generate a playlist with more unknown artist. Go here
